模块二

寻找并锁定最近的敌人

用 Group 全局标记 + 距离算法锁定最近敌人

📋 特性分析 Features
  • 身份标记：代码需要有一种方式从成百上千个节点中瞬间识别出"谁是敌人"。
  • 距离演算：玩家需要遍历所有存活的敌人，并找出几何距离最近的一个。
🔧 配置步骤

打开 Enemy.tscn，在 Node → Groups 面板中为根节点添加标签：

enemies
📷 为敌人添加全局 Group 做判断
📷 利用 enemies 全局 Group 拿到当前游戏中的所有敌人
⚙️ 代码处理 Logic

在 Player.gd 中实现寻找最近敌人的函数：

# Player.gd
func get_nearest_enemy() -> Node2D:
    # 获取场景中所有属于 enemies 组的节点
    var enemies = get_tree().get_nodes_in_group("enemies")
    var nearest_enemy = null
    var min_distance = INF  # 初始设为无穷大

    for enemy in enemies:
        var dist = global_position.distance_to(enemy.global_position)
        if dist < min_distance:
            min_distance = dist
            nearest_enemy = enemy

    return nearest_enemy
📌 老师注记
  • Group 是全局的：无论敌人藏在哪个子节点下，get_nodes_in_group 都能找到
  • INF 的作用：保证第一个敌人一定比初始值更近，被成功记录
  • 敌人全灭时：enemies 数组为空，函数返回 null，调用方需要处理这个情况
