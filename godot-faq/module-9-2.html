模块 9.2

实现 Explosion 爆炸效果

创建爆炸场景、设置爆炸范围、Timer 自动清理、Missile 触发爆炸

🎯 本节目标
  • 创建独立的 Explosion.tscn 场景
  • 导弹 1 秒超时后生成爆炸
  • 爆炸短暂存在后自动删除
  • Missile 负责飞行，Explosion 负责范围效果
🔧 设计分离
Missile → 负责飞行
Explosion → 负责爆炸范围效果

把飞行逻辑和爆炸效果拆成两个场景，便于独立维护和复用。

步骤 11创建 Explosion 场景

在 Godot 编辑器中操作：

Scene 面板 → Other Node → Area2D

重命名根节点为 Explosion，保存为 Explosion.tscn。

步骤 12添加节点

给 Explosion (Area2D) 添加子节点：

Explosion (Area2D) （根节点：负责爆炸范围检测） ├── Sprite2D （显示爆炸图片） ├── CollisionShape2D （爆炸范围） └── Timer （爆炸结束后自动删除）
步骤 13设置爆炸范围

选中 CollisionShape2D：

Inspector → Shape → New CircleShape2D

调整 Radius 控制爆炸范围大小。这个范围决定了爆炸能波及多远。

步骤 14设置 Explosion Timer

选中 Timer，在 Inspector 中配置：

Wait Time  = 0.2
One Shot   = 开启（☑）
Autostart  = 开启（☑）

作用：爆炸短暂存在 0.2 秒后自动删除。

步骤 15连接 Explosion 信号

先给 Explosion 添加一个空脚本（至少包含 extends Area2D），然后依次连接以下两个信号：

15.1 连接 Timer 的 timeout 信号

  • 选中 Timer 节点
  • 切换到 Node 面板
  • 找到 timeout 信号
  • 双击或点击 Connect…，连接到 Explosion.gd

15.2 连接 Area2D 的 body_entered 信号

  • 选中 Explosion (Area2D) 根节点
  • 切换到 Node 面板
  • 找到 body_entered 信号
  • 双击或点击 Connect…，连接到 Explosion.gd

连接成功后，Explosion.gd 会自动生成空的 _on_timer_timeout() 和 _on_body_entered(body) 函数。

步骤 16补充 Explosion 脚本

在 Explosion.gd 中补充两个回调函数的函数体：

extends Area2D

func _on_body_entered(body):
	if body.is_in_group("enemy"):
		body.queue_free()

func _on_timer_timeout():
	queue_free()

作用：爆炸碰到敌人时销毁敌人，0.2 秒后删除 Explosion 自身。

⚠️ 检测不到？先检查 Collision Layer / Mask：
Explosion 的 Mask 必须勾选敌人所在的层；敌人的 Layer 也要正确设置。两者对应上，body_entered 才能触发。

步骤 17修改 Missile 脚本（生成 Explosion）

修改 Missile.gd，加入爆炸场景引用和爆炸函数：

extends Area2D

@export var explosion_scene: PackedScene

var direction = Vector2.RIGHT
var speed = 400

func _physics_process(delta):
	global_position += direction * speed * delta

func explode():
	var explosion = explosion_scene.instantiate()
	explosion.global_position = global_position
	get_parent().add_child(explosion)
	queue_free()

func _on_timer_timeout():
	explode()

流程变化：
导弹飞行 → Timer 结束 → 调用 explode() → 实例化 Explosion → Missile 删除

步骤 18运行效果

点击运行，按 Z 键测试：

按下 Z ↓ 实例化 Missile ↓ 导弹飞行 ↓ 1秒后触发爆炸 ↓ 生成 Explosion ↓ Explosion 自动消失

到这里，学生已经可以测试导弹爆炸效果。

📌 老师注记
  • explosion_scene 的赋值：在编辑器中选中 Missile，将 Explosion.tscn 拖到 Inspector 里的 Explosion Scene 属性中。
  • get_parent().add_child(explosion)：爆炸也要加到 World 层，不能作为 Missile 的子节点，因为 Missile 马上会被删除。
  • global_position 同步：explosion.global_position = global_position 确保爆炸出现在导弹当前位置。
  • queue_free() 顺序：先创建爆炸，再删除导弹，否则导弹位置信息会丢失。
