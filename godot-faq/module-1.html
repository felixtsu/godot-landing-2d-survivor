模块一

设计敌人与生成敌人

用 PackedScene 实例化敌人，掌握随机分布技巧

📋 特性分析 Features
  • 多实例需求：运行时需要同时存在多个敌人，且逻辑一致。
  • 物理特性：需要具备位移能力，能被障碍物（如墙壁）挡住，且能被子弹检测到。
  • 随机性：每个敌人被创建时，位置应在地图范围内随机分布。
🌲 节点树构成 Node Tree

在独立场景 Enemy.tscn 中构建：

CharacterBody2D （根节点：提供物理位移 + 碰撞反馈） ├── Sprite2D （视觉贴图） └── CollisionShape2D （物理边界，通常用 CircleShape2D）
📷 把敌人加入 World 的场景
⚙️ 代码处理 Logic

在主世界 World.gd 中处理生成逻辑：

# World.gd
extends Node2D

@export var enemy_scene: PackedScene  # 引用 Enemy.tscn 文件

func spawn_enemy():
    # 1. 实例化：在内存中创建对象
    var enemy = enemy_scene.instantiate()

    # 2. 坐标计算：设定随机坐标
    var screen_size = get_viewport_rect().size
    enemy.global_position = Vector2(
        randf_range(0, screen_size.x),
        randf_range(0, screen_size.y)
    )

    # 3. 挂载：将其加入场景树，正式激活节点
    add_child(enemy)
📌 老师注记
  • @export var enemy_scene: PackedScene：在 Inspector 面板里拖入 Enemy.tscn 即可关联
  • randf_range：每次调用产生不同的随机浮点数，确保敌人位置不重叠
  • 实例化三步曲： instantiate() → 设位置 → add_child()
