使用计时器定时生成

Timer 定时器控制敌人自动生成节奏

📋 特性分析 Features
  • 固定周期：每隔 X 秒执行一次特定逻辑（如：生成敌人）。
🌲 节点配置
  • 在 World 节点下添加 Timer 子节点
  • Wait Time：设定周期秒数（如 2.0 = 每2秒）
  • Autostart：勾选后场景加载完毕自动开始计时
  • One Shot：不勾选则循环重复触发
⚙️ 代码连接

在 World.gd 中连接 timeout 信号：

# World.gd
func _on_timer_timeout():
    spawn_enemy()  # 每次闹钟响，执行一次生成

💡 信号连接方式：在 Timer 节点的 Node 面板找到 timeout 信号，连接到 World.gd 的 _on_timer_timeout 函数。

📌 老师注记
  • Timer 节点自带时间逻辑：不需要在 _process 里手动计数，直接用信号就行
  • Autostart 勾不勾？如果需要代码控制开始/暂停，不勾，用 timer.start() 和 timer.stop() 控制
  • 暂停敌人生成：调用 $Timer.stop()，恢复时调用 $Timer.start()
