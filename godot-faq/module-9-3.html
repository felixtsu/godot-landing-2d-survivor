模块 9.3

碰到 enemy 提前爆炸

连接 Area2D 碰撞信号、检测 enemy 组、实现命中即爆逻辑

🎯 本节目标
  • 导弹碰到敌人时立即爆炸，不再等到 1 秒超时
  • 使用 body_entered 信号检测碰撞
  • 使用 is_in_group("enemy") 精确筛选敌人
  • 给敌人节点添加 enemy 组
🔍 当前状态 vs 目标状态
目前 → Missile 只会定时爆炸
目标 → 碰到 enemy 立即提前爆炸
步骤 19连接 Missile 碰撞信号

选中 Missile (Area2D) 节点：

  • 切换到 Node 面板
  • 找到 body_entered 或 area_entered 信号
  • 双击或点击 Connect…
  • 连接到 Missile.gd

连接成功后，Missile.gd 会自动生成 _on_body_entered(body) 函数。

步骤 20检测 enemy 组

修改 Missile.gd，在 _on_body_entered 中检测敌人：

func _on_body_entered(body):
	if body.is_in_group("enemy"):
		explode()

逻辑：当 Area2D 检测到刚体进入时，先判断该 body 是否属于 enemy 组。如果是，立即调用 explode() 生成爆炸并销毁自身。

⚠️ 检测不到？先检查 Collision Layer / Mask：
Missile 的 Mask 必须勾选敌人所在的层；敌人的 Layer 也要正确设置。两者对应上，body_entered 才能触发。

步骤 21给敌人添加 enemy 组

选中敌人节点（如 Enemy / Slime / Boss 等）：

Inspector → Node → Groups → 添加 enemy

这样 Missile 才能通过 is_in_group("enemy") 识别该节点为敌人。

⚠️ 注意：Group 名称区分大小写。enemy 与 Enemy 被视为不同组，脚本与编辑器中务必保持一致。

步骤 22运行效果

点击运行，按 Z 键发射导弹，观察两种爆炸触发情况：

按下 Z ↓ Missile 飞行 情况1： 1秒后自动爆炸 情况2： 提前碰到 enemy ↓ 立即爆炸

到这里，导弹已经具备完整的自动销毁和命中爆炸功能。

📦 完整 Missile.gd 参考

整合三个阶段的完整代码：

extends Area2D

@export var explosion_scene: PackedScene

var direction = Vector2.RIGHT
var speed = 400

func _physics_process(delta):
	global_position += direction * speed * delta

func explode():
	var explosion = explosion_scene.instantiate()
	explosion.global_position = global_position
	get_parent().add_child(explosion)
	queue_free()

func _on_timer_timeout():
	explode()

func _on_body_entered(body):
	if body.is_in_group("enemy"):
		explode()
📌 老师注记
  • body_entered vs area_entered：
    如果敌人是 CharacterBody2D / RigidBody2D，用 body_entered；如果敌人也是 Area2D，用 area_entered。
  • Group 大小写敏感：再次强调，is_in_group("enemy") 中的字符串必须与编辑器 Groups 中填写的一致。
  • explode() 复用：超时爆炸和碰撞爆炸都调用同一个 explode() 函数，代码复用性好，便于统一修改爆炸逻辑。
  • 后续扩展：可以在 explode() 中遍历爆炸范围内的所有 body，对敌人造成伤害，而不仅仅是销毁导弹。
