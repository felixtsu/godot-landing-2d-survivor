模块四

让子弹真正打中敌人

Layer/Mask 交叉配置，信号驱动碰撞销毁逻辑

📋 特性分析 Features
  • 感应过滤：子弹应只对敌人层产生反应，忽略玩家。
  • 事件驱动：当重叠发生，由物理引擎通知脚本执行 queue_free()。
🎯 物理层级配置 Collision
Enemy (CharacterBody2D)
Layer → 2，Mask → 3
Bullet (Area2D)
Layer → 3，Mask → 2
📷 Enemy Layer 2 Mask 3，Bullet Layer 3 Mask 2

⚠️ Layer 是"我是什么"，Mask 是"我能碰到什么"。两者数值对应才能触发碰撞。

⚙️ 代码处理 Logic

在 Bullet.gd 中处理 body_entered 信号：

# Bullet.gd
var direction = Vector2.ZERO
var speed = 600

func _process(delta):
    position += direction * speed * delta

# 信号连接至此函数
func _on_area_2d_body_entered(body: Node2D):
    # 再次确认目标身份
    if body.is_in_group("enemies"):
        body.queue_free()   # 销毁敌人
        queue_free()        # 销毁子弹自己
📌 老师注记
  • Layer/Mask 搞反是最高频错误：Enemy Layer=2 Mask=3，Bullet Layer=3 Mask=2（两者 Mask 交叉，Layer 互为对方的 Mask 值）
  • 信号连接方法：在 Area2D 节点的 Node 面板搜索 body_entered，拖拽连接到 Bullet.gd 的函数
  • queue_free()：标记节点在本帧结束后销毁，安全删除，不会影响当次碰撞检测循环
