模块六

TileMapLayer：建造你的世界

用瓦片拼出地图，给墙壁加上物理碰撞

📋 特性分析 Features
  • 瓦片拼图：用小方块（瓦片）拼出大地图，类似 Minecraft / 复古游戏编辑器。
  • 物理碰撞：墙壁需要加 Physics Layer，玩家和敌人都被挡住，不能穿墙。
  • 地板无碰撞：地板只是视觉效果，不需要碰撞。
🌲 节点树构成 Node Tree

在 Main 场景中添加：

Main（Node2D） （游戏主场景） ├── Player （玩家角色） ├── EnemySpawner （敌人生成器） └── TileMapLayer （地图层，注意：不是 TileMap）

⚠️ Godot 4 用 TileMapLayer，不是 TileMap（TileMap 是 Godot 3 的旧名字）

⚙️ 操作步骤 Step by Step

第一步：导入瓦片图片

⬇️ 打开瓦片图 kenney_tileset_spritesheet.png（随站点一同部署；在新标签页打开后，用浏览器「另存为」保存到本地，再拖入 Godot）

1. 把 kenney_tileset_spritesheet.png（16×16 像素，7列×3行，共21块瓦片）拖入 FileSystem 面板
2. 图片会被导入为资源

第二步：创建 TileMapLayer

1. 在 Main 场景右键 → 添加子节点
2. 搜索 "TileMapLayer" → 创建
3. Inspector → Tile Set → 点击 "New TileSet"
4. 底部会打开 TileSet 编辑器面板
📷 创建 TileMapLayer 节点

第三步：把图片变成瓦片

1. 在 TileSet 编辑器中，点击左侧 + 按钮新建
2. 把 FileSystem 里的 kenney_tileset_spritesheet.png 拖入编辑器空白区域
3. 系统会弹出切片设置：将 Columns 设为 7，Canvas 保持默认，确认即自动切片
📷 把瓦片图片导入为 TileSet Atlas

第四步：给墙壁加碰撞（Physics Layer）

1. 在 TileSet 编辑器展开 "Physics Layers"
2. 点击 "Add Element"（会看到 "Physics Layer 0" = 第一层）
3. 切换到 Paint（画笔）工具
4. 在左边的瓦片列表中，选中有墙壁图案的瓦片
5. 在编辑区域点击墙壁图块 → 涂上 Physics Layer 0 标记
6. 现在这个瓦片就有物理碰撞了！
📷 在 TileSet 编辑器中为墙壁瓦片标记 Physics Layer 0

第五步：在场景里画地图

1. 画地图前，先把编辑器左下角切换到 2D Tab（不是 3D 视图）
2. 在右侧 Scene 面板选中 TileMapLayer 节点
3. 在顶部工具栏选择 Brush（画笔）工具
4. 选择地板瓦片 → 在编辑区画地板（随便画）
5. 选择墙壁瓦片 → 画墙壁（这些墙是真正挡人的）
6. 双击已画的瓦片可以擦除（取消该格）
7. F5 测试：玩家和敌人都被墙挡住 = 成功！
📷 在 2D 编辑视图中用 Brush 工具绘制地图
📌 老师注记
  • Physics Layer 没 Add Element 就画 = 墙壁穿人！学生最常忘记的一步
  • TileMapLayer vs TileMap：Godot 4 新名字是 TileMapLayer，老名字 TileMap 仍然可用但已过时
  • 地板不需要 Physics Layer：只有墙要加碰撞标记
  • TileSet 里有碰撞标记的瓦片：CharacterBody2D（如 Player、Enemy）会自动被挡住
💡 核心概念
  • TileSet = 瓦片资源包：定义每块瓦片长什么样、有没有碰撞
  • TileMapLayer = 画布：在场景里用瓦片画画
  • Physics Layer 0 = 墙壁标记：有这个标记的瓦片 = 会被物理对象挡住
