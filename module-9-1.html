模块 9.1

🎯 实现 Missile 导弹飞行

创建导弹场景、加节点、玩家按 Z 键发射、1 秒后自动销毁

🎯 本节目标
  • 创建独立的 Missile.tscn 场景
  • 导弹发射后沿固定方向自动飞行
  • 1 秒后自动销毁，防止内存泄漏
  • 玩家按 Z 键即可发射导弹
🌲 节点树构成

新建 Missile.tscn，根节点 Missile (Area2D)：

Missile (Area2D) （根节点：负责碰撞检测与飞行逻辑） ├── Sprite2D （显示导弹图片） ├── CollisionShape2D （导弹碰撞区域） └── Timer （控制导弹存在时间）
⚙️ 操作步骤 Step by Step

步骤 1：创建 Missile 场景

在 Godot 编辑器中操作（二选一）：

  • Scene 面板 → 右键 → New Inherited Scene
  • 或：Scene 面板 → Other Node → Area2D

重命名根节点为 Missile，保存为 Missile.tscn。

步骤 2：添加子节点

给 Missile (Area2D) 添加三个子节点：

  • Sprite2D —— 显示导弹图片
  • CollisionShape2D —— 导弹碰撞区域
  • Timer —— 控制导弹存在时间

步骤 3：设置 Sprite2D 图片

  • 在 Inspector 中找到 Texture
  • 拖入导弹素材图片
  • 调整 Transform → Scale 让导弹大小合适

步骤 4：设置碰撞区域

  • Inspector → Shape → New RectangleShape2D（或 New CircleShape2D）
  • 调整 Shape 的大小，覆盖导弹图片范围即可

步骤 5：设置 Timer

选中 Timer，在 Inspector 中配置：

Wait Time  = 1
One Shot   = 开启（☑）   # 只触发一次
Autostart  = 开启（☑）   # 自动开始计时

作用：导弹飞行 1 秒后自动销毁。

步骤 6：连接 Missile Timer 信号

先给 Missile 添加一个空脚本（至少包含 extends Area2D），然后选中 Timer 节点：

  • 切换到 Node 面板
  • 找到 timeout 信号
  • 双击或点击 Connect…
  • 连接到 Missile.gd

连接成功后，Missile.gd 会自动生成空的 _on_timer_timeout() 函数。

步骤 7：补充 Missile 脚本

在 Missile.gd 中补充变量、飞行逻辑和回调函数体：

extends Area2D

var direction = Vector2.RIGHT   # 飞行方向（默认向右）
var speed = 400                 # 飞行速度 400 像素/秒

func _physics_process(delta):
    global_position += direction * speed * delta

func _on_timer_timeout():
    queue_free()                # 1 秒后自动删除导弹

步骤 8：玩家发射 Missile

在 Player.gd 中添加发射逻辑：

@export var missile_scene: PackedScene    # 在 Inspector 把 Missile.tscn 拖进来

func _input(event):
    if event.is_action_pressed("fire_missile"):
        fire_missile()

func fire_missile():
    var missile = missile_scene.instantiate()
    missile.global_position = global_position
    get_parent().add_child(missile)

⚠️ 关键点：get_parent().add_child(missile) 必须把导弹加到 World 层，而不是 Player 自己。如果加到 Player 下，导弹会跟着玩家一起移动！

步骤 9：绑定输入（Z 键）

打开项目输入设置：Project → Project Settings → Input Map，添加新动作 fire_missile，绑定 Z 键。

步骤 10：运行效果

按下 Z
  ↓
实例化 Missile
  ↓
导弹飞行（direction * speed）
  ↓
1秒后自动销毁

到这里，学生已经可以测试导弹发射与飞行效果。

📌 老师注记
  • global_position vs position：发射时用 global_position 确保坐标系一致，避免父子节点坐标叠加导致偏移。
  • add_child 到 parent：如果加到 Player 节点下，导弹会跟着 Player 一起移动，无法实现独立飞行。
  • Timer 配置：One Shot 和 Autostart 都要开启，否则计时器不会自动工作。
  • 方向调整：如需让导弹朝玩家面向方向飞行，需在 Player 中记录最后朝向 last_dir：
# Player.gd
var last_dir = Vector2.RIGHT

func _physics_process(delta):
    var direction = Input.get_vector("ui_left", "ui_right", "ui_up", "ui_down")
    if direction != Vector2.ZERO:
        last_dir = direction

func fire_missile():
    ...
    missile.direction = last_dir

🚀 导弹会飞了，但撞到东西没有反应
