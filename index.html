模块七

模块八

给游戏加上分数显示，掌握展示层与数据层分离的概念，Bullet 消灭敌人时自动计分。

❓ 常见问题 FAQ

  • 如何提交 Godot Web 作品？
    先用课堂账号登录，打开 作品上传页， 将 Godot 导出的 Web 目录整体压缩为 zip，填写作品名后提交，等待老师审核。

📋 老师注记

以下两点是学生最容易出错的地方，讲课时可以特别提醒：

  • 重点检查 Layer / Mask 的对应关系，这是碰撞失效最常见的原因
  • 确保子弹发射时 add_child 到了 World 节点，而不是 Player 自己
  • TileMapLayer 的 Physics Layer 没 Add Element 就画 = 墙壁穿人！
  • Camera2D 要勾 Enabled 属性才能生效
  • Process Mode = Always 节点上的属性在 Inspector 顶部，没设 = 暂停时 UI 也停
